蘿蔔頭聊天室用法

1. 🔐 註冊帳號

請輸入以下三項：

- Email（作為登入帳號）

- 密碼（用來登入）

- Line ID（你對朋友顯示的暱稱）

→ 輸入完成後，請按下「註冊」。

2. 🔓 登入聊天室

輸入你剛才註冊的 Email 和密碼，按下「登入」即可進入聊天室畫面。

3. 👤 加好友

- 請輸入對方的 Line ID

- 按下「加好友」按鈕

- 若對方存在，就會成功加入好友列表！

4. 👥 建立群組

- 在「群組名稱」欄輸入想要的群組名稱

- 按下「建立群組」即可建立

- 建立者會自動成為第一位成員

5. ✉️ 聊天功能

- 點選好友或群組名稱

- 下方會出現聊天區塊

- 輸入訊息後按「送出」，即可聊天

6. 🚪 登出帳號

按下「登出」按鈕即可登出，畫面會回到登入畫面。

📌 注意：目前聊天室尚未支援上傳圖片或貼圖，但已支援多對多聊天、一對一聊天與好友功能。
